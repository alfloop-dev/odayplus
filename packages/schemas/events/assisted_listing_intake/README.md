# Assisted Listing Intake Event Schemas

This package defines and documents the versioned event schemas for the Assisted Listing Intake boundary.

The schemas are defined and parsed from the central contracts:
- `docs/events/ODAY_PLUS_ASSISTED_LISTING_INTAKE_EVENTS_V1.yaml`
- `docs/events/ODAY_PLUS_ASSISTED_LISTING_INTAKE_EVENTS_V1_1_ADDENDUM.yaml`
- `docs/events/ODAY_PLUS_ASSISTED_LISTING_INTAKE_EVENT_PAYLOAD_SCHEMAS_V1.yaml`

Refer to these files for the normative definitions of all event types, aggregate bindings, partition keys, and JSON Schema payload schemas.
